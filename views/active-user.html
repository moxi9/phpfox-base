Hi {{ name }}!

The name above is taken from the PHP variable we assigned in the start.php file. Look for 'name' => $Controller->active->name

When you are in a controller the $Controller->active holds the object of the active user that is logged in.

Using Twig Variables - Hi {{ ActiveUser.name }}!

You can also use our Twig variables to get the active user. The ActiveUser. prefix in Twig variables are reserved for users that are logged in.
